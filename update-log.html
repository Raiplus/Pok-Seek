📝 Update Logs – Version 0.0.1

Release Date: 2025-05-09

Developer: Rishabh Singh Rajawat (aka Raiplus)

✅ Features Added

  • 🎮 Core game logic for Pokémon guessing implemented.
  • 🖼️ Image reveal on correct selection with unblur effect.
  • ✅ Correct answer turns green, wrong turns red.
  • 🧠 Scoring system integrated (+1 for correct guess).
  • ⚡ Fast-guess bonus system (up to +5 points).
  • 🔒 UI disabled after one guess to prevent spamming.
  • 🔁 Play Again button state added.
  • 📊 Score visible and updates in real-time.
